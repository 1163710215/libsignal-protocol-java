Class Overview

The main entry point for axolotl group encrypt/decrypt operations. Once a session has been established with GroupSessionBuilder and a SenderKeyDistributionMessage has been distributed to each member of the group, this class can be used for all subsequent encrypt/decrypt operations within that session (ie: until group membership changes).
